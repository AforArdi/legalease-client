## 📖 Purpose

Finding the right legal representation can be an overwhelming and opaque process. **LegalEase** simplifies this journey by providing a centralized, transparent, and highly secure platform. Clients can easily browse legal experts by specialty or consultation fee, send hiring requests, manage transactions, and leave authenticated reviews. At the same time, lawyers and administrators are equipped with dedicated, role-based dashboards to effortlessly manage their profiles, review requests, and oversee platform analytics.

---

## ✨ Key Features

**[Feature 1: Seamless Lawyer Discovery & Hiring]**
* **Problem Solved:** Clients often struggle to find specialized lawyers within their budget and availability constraints.
* **Tech Used:** Next.js App Router, React Server Components, and optimized API routing.
* **Impact:** Empowers users to confidently filter legal experts by category, fee range, and status in real-time, reducing the time required to find the perfect representation from days to minutes.

**[Feature 2: Secure Role-Based Portals]**
* **Problem Solved:** Different user types (Clients, Lawyers, Admins) need entirely different tools without a cluttered, confusing UI.
* **Tech Used:** Better Auth for robust session management and HeroUI for highly polished, distinct interface components.
* **Impact:** Provides dedicated, protected dashboards for every user role. Lawyers can manage incoming requests and their public profile, while admins have full visibility over transactions, user roles, and platform metrics, ensuring streamlined operations and data privacy.

**[Feature 3: Transparent Transactions & Authenticated Reviews]**
* **Problem Solved:** Lack of trust in online legal services and fake reviews.
* **Tech Used:** Stripe integration for payments and strict, server-verified MongoDB logic.
* **Impact:** Creates a highly trustworthy ecosystem where payments are securely handled. Most importantly, the platform employs strict validation ensuring that *only* clients who have successfully hired and paid a specific lawyer are granted the ability to leave a public review.

## 🛠 Tech Stack

**[ Frontend | Backend | Database | Others ]**

* **Frontend:** Next.js 16 (App Router), React 19, Tailwind CSS v4, Framer Motion
* **UI Framework:** HeroUI (formerly NextUI)
* **Backend Integration:** Express.js REST API (Refer to the Backend Repo)
* **Database:** MongoDB
* **Authentication:** Better Auth
* **Payments:** Stripe

### 📦 Key NPM Packages Used
* `next`, `react`, `react-dom` - Core framework and UI rendering
* `@heroui/react`, `@heroui/styles`, `@heroui/input` - UI Component Library
* `better-auth`, `@better-auth/mongo-adapter` - Secure authentication handling
* `@stripe/stripe-js`, `stripe` - Payment processing
* `motion` - Smooth page and element animations
* `react-hook-form` - Form state management and validation
* `react-hot-toast` - Elegant notification toasts
* `lucide-react`, `react-icons`, `@gravity-ui/icons` - Comprehensive icon sets
* `swiper` - Responsive image and content sliders
